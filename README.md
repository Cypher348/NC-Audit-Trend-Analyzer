#NC Audit Trend Analyzer 📊
An automated data analysis pipeline to track, visualize, and analyze Non-Conformity (NC) records from quality audits. This project helps quality assurance teams identify recurring issues, high-risk stations, and month-over-month trends.

🚀 Key Features
Trend Visualization: Monthly line charts to monitor NC spikes.

Severity Distribution: Pie charts showing the breakdown of Critical vs. Minor issues.

Station Performance: Identification of the Top 10 stations with the highest NC counts.

Automated Preprocessing: Handles missing values and date formatting using Python/Pandas.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, Matplotlib, Seaborn, NumPy

Platform: Google Colab / Jupyter Notebook .
